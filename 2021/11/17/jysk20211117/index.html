如何辅导小孩子学习？

发表于 2021-11-17 更新于 2021-11-30 分类于 教育.思考 本文字数： 3.6k 阅读时长 ≈ 3 分钟

问题

小孩，目前是在读三年级。在这三年中，我观察了自家小孩的学习情况：

  • 不重视复习，尤其好像看不懂数学课本。
  • 同样类型的题目，有时会做有时不会做。
  • 知识学得不牢固，理解不是很深刻。
  • 语文，写字有时还是写拼音。
  • 运动能力比较弱，比如不会正确地跑步、跳绳。
  • 写字不够工整。
  • 生活常识较缺乏。
  • 学习习惯不好，有点拖拉。
  • 不爱思考，遇到较难的题目，就懵了。
  • 不会举一反三。

学科分析：

学科 优势 弱势
语文 1、基础良好。
2、阅读理解一般。
3、作文良好。		 1、反问句不太熟悉。
2、关联词不太熟悉。
3、课文理解不够。
4、作文段落不够分明。
数学 1、基础一般。 1、作业不够工整。
2、题目不会举一反三。
3、数学课本看不懂。
4、没有形成简单的数学思维。
英语 1、基础一般。
2、听力良好		 1、学习表现不是很稳定。
2、英语阅读理解弱。
科学 1、一般 1、缺乏自然常识。

认知

​ 辅导小孩学习要掌握方法，不能以大人的认知去辅导他。在大人的认知里，小学知识是很简单的，毕竟大人的认识是由于阅历的丰富、教育而被提升的。以大人的认知去辅导小孩，他不一定能够接受得住，所以要讲究方法。

​ 那我们如何发现和评估小孩的认知水平呢？先倾听小孩的认识，很多时候，我们总是喜欢将自己的认知、方法灌输给小孩，这无异于揠苗助长，倾听是打开了解小孩认知的大门。我们不仅要倾听，还要观察，要观察他们是怎么做的。

​ 总之，要先了解小孩的认知，必须听他们怎么说、看他们怎么做，然后对比彼此认知的差异，接着要提升小孩的认知。

​ 提升小孩的认知，首先要强化他对知识的了解。那如何强化呢？我尝试着才有对比法：

  • 先记录小孩的想法。

  • 再介绍自己的想法。

  • 对比两种想法的差距。

  • 给小孩讲解大人的想法。

  • 让小孩讲解新学到的想法。

    这样的方式，看能不能提高小孩的认知。当然提升认知的方法要不断摸索和总结。

方法

根本方法

  • 耐心
  • 由易到难
  • 举一反三，类比。

具体的方法

培养良好的学习习惯

  1. 培养遵循：预习、复习、练习的学习方式。
  2. 写字要工整。
  3. 做练习的时候，准备好相应的学习文具，比如课本、练习本、草稿子、笔、橡皮擦、尺子。
  4. 利用番茄学习法，集中45分钟，休息5分钟模式。
  5. 做笔记：课堂笔记、错误题笔记、课外阅读笔记等。
  6. 错开时间、穿插复习（按3、7、15天 间隔性复习知识）。
  7. 培养不骄傲、不得瑟的习惯。

教会小孩学会复习

  1. 先让小孩按科目回忆今天的学习内容，比如回忆概念、事实、事件。
  2. 然后小孩讲解给你听(小孩可以自己复述，采用录音机)。
  3. 让小孩再次看课本，再重复步骤。
  4. 纠正小孩的错误（自己纠正错误）。
  5. 重复步骤3.

考试

虽然，我们不提倡题海战术，但是适当的检测有助于发现小朋友知识掌握情况。而且，考试并不完全是通过试卷，比如：

  1. 听写词语（单词）。
  2. 回忆背诵课文。
  3. 提问。
  4. 让小朋友对课文进行概述。
  5. 小朋友的自我检测（比如自己 回忆今天学了哪些知识：生词、词语、概念、事实、事件）。

成长性思维

辅导小孩，最终是要让小孩具备成长性思维。要培养成长性思维，首先要进行多样化的练习，比如单学科、多学科的多样性练习，在这个过程中，辅导小孩，进而学会：

  1. 对比、类比思维。
  2. 举一反三思维。
  3. 归纳思维。
  4. 发散思维。
  5. 交叉思维（格栅思维）。

学科方法表

学科 方法
语文 1、辅导造句
2、辅导概述课文

2、辅导 通过对话，进行英语学习。
3、辅导 小孩进行简单的英语阅读理解。
4、辅导 小孩进行简单的英语写作。

打造心智模型

成功的秘诀在于你的内心。如果你在思考问题时持批判态度，。。。。。。，通过培养自己的约束力和勇气，你就不会让他人的情绪波动来左右你的目标。说到底，你的投资方式远不如你的行为方式重要。（改编自<<聪明的投资者>>）。

这里的行为模式，就是所说的心智模型。每个人，在实际生活中，会慢慢培养自己的心智模型：

  1. 思维模式。

  2. 情绪反应。

  3. 行动。

  4. 试错方法。

遇到困难（未知、艰巨、。。。），个人的反应取决于各自的心智模型：

  1. 情绪反应：乐观的人，则情绪波动不是很大；悲观的人，则满脸写着忧愁。
  2. 行动：积极的人，虽然有不同的情绪反应，但还是会启动自己的思维模型进行问题分解、尝试解决；消极的人，则唉声叹气，希冀别人帮他解决。
  3. 思维模式：我们解决问题采取的方式、步骤。有的人，采用分治方法，将问题分解小问题、主次问题，再各个解决；有的人尝试利用曾经掌握过的方法进行尝试解决。有的人会尝试多种方式进行解决；有的人尝试拿一把“锤子”解决所有的问题。
  4. 试错方法：有的小孩学会快速试错，有的小孩学会想得很仔细再行动。

而我们协助小孩打造心智模型，就是为了帮忙小孩在学习过程中能够：

  1. 积极、乐观地学习。
  2. 能够有步骤地学习、解决问题：比如尝试学过的知识来理解新的知识，再用新的知识去解决旧问题，并能掌握发现问题、暴露缺陷的方法。
  3. 能够利用多种思维方式进行解决问题（类比、发散、归纳、表格、图形、交叉）。
  4. 乐观、谦虚、想像、求证的心态。

全面发展

曾经跟同事交流，有的同事想培养小孩的编程能力，觉得IT工作是一项不错的工作，而且外面有很多培训机构都打着培养小孩思维的幌子进行编程培训，所以他觉得很有必要让小孩尽早接触编程。正如学英语，70、80后的人，学英语是在初中开始学，而且也是正常地学，但也培养了很多需要外语进行工作的人，同时随着翻译设备的发展，外语在未来的工作中，显得不是那么重要。其实，编程只是一种逻辑思维，也许在20世纪，逻辑思维强的人，确实可以发展很好。但是，逻辑思维只是一种规律思维，在未来很容易被替换，比如会计工作，就因为各种财务软件，IT系统，显得不是那么重要的，尤其数字化转型，主要根据数据格式，将数据导入各种相应的IT系统，会计报表就很容易出来。那么未来什么最重要，先看一种图：

拥有高智商，不如拥有高情商。高概念、高感性能力，正从生活的边缘走向生活的中心。生活的意义，是一种新的财富。

知识，不再是关键的力量（图和文字摘自<<全新思维>>）。

要在这个时代求生存，任何人可能要进行灵魂拷问：

  1. 完成同样一件工作，外包人员是不是比我的成本低。比如工业时代，发达国家进行产业转移，把制造业转移到发展中国家，造成发达国家的工人几乎失业。在信息时代，发达国家把信息工作转移到印度、中国，尤其印度，造成印度的软件外包特发达。
  2. 电脑是不是比更迅速？很多手工、机械、重复的工作，很快都会被IT系统给取代，比如会计。
  3. 在物质财富极大充裕的时代，我提供的服务是不是人们所需要的？ 这也是为什么有些网红会大赚特赚，在早些时候，这些人正如社会上耍嘴皮的人，但是5G、移动APP发展的时代，这些人就有发挥的舞台。人，只要有舞台，迟早会一飞冲天。

所以，未来创造思维和创造性工作将从职场的边缘走向职场的中心。那什么是创造性工作呢？创造性包含创建艺术美和情感美，找到自己的快乐并感染他人。比如：

  1. 陶艺创作。
  2. 漫画创作。
  3. 音乐创作。
  4. 短视频创作。
  5. 文学创作。

​ 所以，小孩不要过早地受限学习。有些能力是成长性的，需要尽早投入持续培养；有些能力只需要阶段性培养的 。从德、智、体的角度来看，体即运动能力，运动之所以重要，取决于是否持续拥有“革命的根本”。德，广义的德，应该是一个人的品性，也是要从小去引导的，不排除浪子回头的可能性，但是良好的品性是小孩生活幸福和开心的关键。智，我的理解是小孩解决问题的思维能力和行为习惯，也是要从小去引导的。智的训练，类似STEAM（S（科学）T（技术）E（工程）A（艺术）M（数学））的训练，有条件的话也可以进行经济素养训练。为什么智的训练，类似STEAM的训练，因为STEAM体系涵盖了科学、技术、工程、数学素养，STEM能力强的人，未来职场发展也大概率很成功，不仅拥有胜任逻辑性质的工作，还拥有创作能力，不过这创作能力偏理工性的。那么A呢？A是用来加强个人的人文、艺术情操，强化创作能力，即强化理工性创造的想象力，又补充了感性相关的创造能力。

​ 所以，德智（STEAM）体能力，是从小就要持续投入培养的，这些能力是后续各种能力的基石。这种能力培养得好，小孩的未来各种可能性相对来说比较多，用《论语·为政》中的话来说：君子不器。如果过早地培养类似编程能力，而忽视基石、通用能力，小孩的未来相对来说缺乏一定的柔性，比较固定。当然也许编程能力，会培养一些STEM能力，但总感觉不宜过早投入。

​ 天之道，损有余而补不足，人之道，则不然，损不足以奉有余，小孩的时间是固定的，本来培养通用、基础能力，都已经不足的，还不合时宜地培养非必要的能力，显得有点“损不足以奉有余。”
